CSS란?

HTML이 웹 문서의 구조를 담당한다면
CSS 글꼴, 색상, 여백 같은 스타일을 추가하여
보기 좋게 꾸며주는 역할을 담당한다.

적용방법

  • 내부 스타일 , style태그
  • 외부 스타일

선택자

  • 전체 선택자 *
  • 태그선택자 tag
  • 클래스선택자 .
  • 아이디선택자 #
  • 자손선택자 공백(　) : li .orange
  • 자식선택자 >
  • 인접형제선택자 +
  • 인접형제선택자 ~
  • 일치선택자 li.orange
  • 가상클래스선택자 :hover :active :focus
  • 가상요소선택자 :: before , ::after
